## Sobre este projeto

Este é um fork do **OBS Studio** adaptado para Termux/Android.  
O projeto foi totalmente gerado usando IA e requer que o usuário faça **ajustes manuais**, incluindo:

- Configurações de compilação específicas para Android/Termux  
- Ajustes de dependências e paths  
- Modificações de código para compatibilidade com dispositivos móveis

O OBS Studio original pode ser encontrado em [https://obsproject.com](https://obsproject.com).
